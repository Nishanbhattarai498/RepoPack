# repopack

[![PyPI version](https://img.shields.io/pypi/v/repopack-tool.svg)](https://pypi.org/project/repopack-tool/)

Repopack is a lightweight Python library that packages an entire project into a single, readable markdown bundle. It walks the project tree, skips common build and virtual environment folders, and writes each included source file into a structured markdown document for documentation, sharing, and AI-assisted workflows.
